1. 概要 OVERVIEW

本報告は、10年前に発生した「スミス家夫婦殺害事件」に関する再捜査の最終結果を記録したものである。新たに押収された証拠および関係者の供述の精査により、クロエ・デイビスとリアム・オコネルの共犯関与が立証され、両名は逮捕された。

2. 証拠分析 EVIDENCE ANALYSIS

  • ソフィア・トンプソン宅のガレージに保管されていた段ボールを押収。内部から血痕の付いた衣類数点、分解状態の拳銃、及びナイフを発見。
  • DNA鑑定の結果、血痕はスミス夫妻のものと一致。
  • 電気使用量記録（8月25日19時の異常値）は、遺体冷却のために冷房が強稼働されたことを示す。
  • 水道使用量記録（8月25日20時の異常値）は、浴槽での流水および氷投入による偽装工作を示す。
  • 関係者のアリバイ表との照合により、複数の虚偽供述が判明。

3. 供述要約 STATEMENT SUMMARY

8月25日 19時頃（事件当日）
クロエ・デイビスがマイケル・スミスを銃で殺害。冷房を強く稼働させ遺体を冷却し、その後カフェでアリバイを構築した。同時刻、リアム・オコネルは第三者との買い物によりアリバイを確保していた。

8月25日 20時頃
エミリー・スミスが帰宅。リアム・オコネルはナイフで殺害後、遺体を浴槽で冷却し、死亡推定時刻を不明瞭にした。

同日 21時〜23時頃
両名は映画館に赴き、互いのアリバイを補強。終了後に路上で証拠品の入った段ボールをリアムからクロエへ引き渡した。

8月27日 朝7時頃
クロエ・デイビスが遠隔操作によりエアコンを停止。腐敗を進行させ、発見時の捜査を混乱させた。

4. 結語 CONCLUSION

動機は遺産の独占を目的としたものであり、犯行後には巧妙な工作が行われていた。また、逮捕時にクロエ・デイビスがレベッカ捜査官に負傷を負わせたため、加重処罰が適用される見込みである。
